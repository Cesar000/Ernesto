INVESTIGACION DE CONCEPTOS


La propiedad position de CSS especifica como se posicionar un elemento en el documento. las propiedades top, right, bottom, y left determinan la localizacion final de los elementos posicionad



width especifica la anchura del area de contenido de un elemento. El área de contenido está dentro del padding, borde, y margen del elemento.



La propiedad CSS height especifica la altura del area de contenido de un elemento. El área de contenido está dentro del padding, borde, y margen del elemento.



La propiedad margin es una de las "propiedades shorthand" que define CSS y que se utilizan para establecer de forma abreviada el valor de una o más propiedades individuales



La propiedad CSS padding establece el espacio de relleno requerido por todos los lados de un elemento. El área de padding es el espacio entre el contenido del elemento y su borde ( border ) no se permiten valores negativos.
